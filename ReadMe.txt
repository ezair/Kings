Description:
	This is a python2 script written for the purpose of
	being able to play the card game, kings on your
	android phone (or pretty much any platform).
	The program is fairly straight forward.
	The program is completed as well (no more updates).

IMPORTANT NOTE:
	Do NOT move the sound folder's location or
	the sounds inside of it (unless you want to replace them).

Programming Language:
	Python2

Requirements:
	Must have the following installed:
		Python2
		pyttsx: API for text to voice in python.
				(pip install pyttsx)
		pygame: Used for sound.
				(pip install pygame)
Platforms:
	Windows 7, 10
	MacOS
	Linux
	Android
